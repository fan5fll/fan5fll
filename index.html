Vinny's programming tips.

Steps when starting a new program

  1. Open up a new project file
  2. Save the project - Use a helpful naming convention
  3. Our team will use this: mission_name_v{version}
  4. Make sure to put the files in the same place every time. For example, always put the files in an ev3 folder in My Documents.
  5. Build away!
  6. Save as often as possible.
  7. Don't be afraid to move fast and break things!
